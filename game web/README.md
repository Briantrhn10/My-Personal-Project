# 🧠 Memory Match Game

Sebuah permainan asah otak interaktif di mana pemain harus menemukan semua pasangan kartu yang cocok. 

Proyek ini dikembangkan secara spesifik untuk mengimplementasikan logika permainan dasar, manajemen *state*, dan merancang antarmuka pengguna (UI) yang responsif. Melalui proyek ini, fokus utamanya adalah memastikan interaksi yang mulus antara *input* dari pengguna dan respons dari sistem.

**Fitur Utama:**
* **Logika Pencocokan Kartu:** Sistem di balik layar yang melacak kartu yang sedang dibalik dan memvalidasi apakah keduanya cocok.
* **Interaksi Dinamis:** Umpan balik visual secara langsung saat kartu dibalik, cocok, atau gagal dicocokkan.
* **Desain Antarmuka:** Tata letak yang rapi dan mudah dimainkan.
